# Docker Image for Slackcat

## What is Slackcat

Upload files and snippets to Slack chat from the command-line.

How to use this image

```
 echo hello | docker run -i codecentric/slackcat:0.2.5.1 --token <insert_token_here> -c bigdatamunich
```
